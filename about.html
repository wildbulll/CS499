Over the course of my Computer Science program at Southern New Hampshire University, I have developed practical skills and a deeper understanding of software design, algorithms, database systems, and secure programming practices. This self-assessment reflects on how my work in this course and the program as a whole demonstrates my ability to meet the four SNHU Computer Science program outcomes, which are:

1. Design and evaluate computing solutions that solve problems and meet the requirements of varied users

Throughout the program, I’ve worked on projects that involved creating software to meet real user needs. For example, my Animal Rescue Dashboard (available on my portfolio) is a web application that allows users to browse and filter animals using an interactive interface. These updates demonstrate my ability to build meaningful and user-centered computing solutions.
